# 🌟 Motivational Quote Generator App (Flutter)

## 📌 Project Description

The **Motivational Quote Generator App** is a simple and interactive Flutter application that displays inspirational quotes to users. The app allows users to generate random motivational quotes, view all stored quotes, add new quotes, and delete unwanted quotes.

This project demonstrates the use of:

* Stateful Widgets
* Navigation between screens
* List management in Flutter
* Random quote generation
* Basic UI design with gradients and cards

---

## 🚀 Features

✅ Generate a Random Quote
✅ View All Stored Quotes
✅ Add New Quotes
✅ Delete Existing Quotes
✅ Beautiful Gradient UI Design
✅ Separate Screen for Quote Management

---

## 🛠️ Technologies Used

* **Flutter**
* **Dart**
* Material Design Widgets

---

## 📱 App Workflow

1. When the app starts, the user sees a welcome message.
2. Pressing the **"Generate Quote"** button displays a random motivational quote.
3. Clicking the **List icon** in the AppBar opens the "All Quotes" screen.
4. On the Quote List screen:

   * Users can add a new quote.
   * Users can delete existing quotes.
   * All quotes are displayed in a scrollable list.
